# Kavach - Women Safety App

Kavach is a mobile-first women safety application that provides real-time SOS alerts, safe route planning, anonymous incident reporting, and AI-powered safety assistance.

---

## Features

- One-tap SOS with emergency contact alerts and siren
- Safe Journey with Google Maps route safety scoring
- Voice activation — say "Kavach" to trigger SOS
- Anonymous incident reporting
- Emergency helplines with tap-to-call
- Evidence vault for secure media storage
- AI chatbot for safety guidance
- Real-time location sharing

---

## Tech Stack

- **Frontend:** React.js
- **Backend:** Spring Boot (Java)
- **Database:** MySQL
- **APIs:** Google Maps API, Anthropic Claude AI

---

## Getting Started

```bash
cd kavach-frontend
npm install
npm start
```

Create a `.env` file in `kavach-frontend/`:
REACT_APP_GOOGLE_MAPS_KEY=your_google_maps_key_here
